1. Create this folder structure:

project/
├── app.py
├── requirements.txt
├── templates/
│   └── index.html
└── static/
    ├── app.js
    └── styles.css

2. Install packages:
   pip install -r requirements.txt

3. In app.py, change DB_CONFIG password and database name if needed.

4. Run:
   python app.py

5. Open:
   http://127.0.0.1:5000
